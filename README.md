# awesomemen.github.io

记录一些很赞的日志。

Personal technical homepage built with Hugo and deployed with GitHub Pages.

## Local development

1. Install Hugo Extended.
2. Run `hugo server`.
3. Open the local URL printed by Hugo.

## Build

Run `hugo --gc --minify`.

## Deploy

Push to `main`. GitHub Actions builds the site and publishes it to GitHub Pages.
